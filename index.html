Conversion tips

You can convert now Personally. WOW!

         
  html,
  body {
      font-size: 16px;
      /* This is Root font-size */
  }

  p {
      font-size: 18px;
  }

  p {
      font-size: px / root-font-size;  /* This is Constructor */
  }

  p {
      font-size: 1.125rem;
      /* same as 18px */
  }
